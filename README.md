# 概述

这是一个帮助中学生快速计算相对分子质量的工具。

因为我每次遇到计算题都要去翻课本，然后按计算器，不胜其烦，导致了这个库的出现。

# 使用方法

首先安装依赖：

```bash
$ pip install -r requirements.txt
```

然后直接运行 `main.py` 就行了，输入 `exit` 或 `quit` 退出：

```bash
$ python main.py
> NaOH
40
> H2SO4
98
> (NH4)2SO4
132
> exit
```
